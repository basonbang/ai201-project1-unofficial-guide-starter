  → here is a list of the topics that are taught in the classroom:

Summary of Topics/Syllabus
1. Fundamentals of Multimedia
- Introduction to Multimedia systems. Elements of Multimedia: Audio, Video, Images, Graphics, and Text.
- Sensors and multimedia capture
- Main data types and characteristics: Text, Audio, Image, and Video.
- Non-Standard media
- Introduction to media processing and analysis.
2. Media Processing and Analysis
- Text Processing and Analysis: Latent Semantic Analysis, TFIDF
- Introduction to Wordnet
- Image and Signal Processing
- Audio Processing and Analysis
3. Multimedia Transmission
- Encoding and Compression
4. Storage, Organization, and Access
- Storage systems
- Media data modeling, Unified multimedia data modeling, and semantics of multimedia
- Unified Multimedia indexing
- Multimedia storage, query, and retrieval: Multimedia databases, information retrieval (IR), IR on the WWW, Web Search
5. Multimedia UI design
- Quality of Experience (QOE) and Experiential Computing
- Experiential UIs
6. Multimedia Description Frameworks
- MPEG
7. Applications
- Multimedia creation and presentation environments
- Multimedia in life-sciences and medicine
- Multimedia information on WWW
- Sensor Networks
- Ubiquitous computing
- Multi-sensor autonomous robotics
- Personal information management
